# install-fe-env

快速安装前端环境脚本。

## 安装

### Mac 系统

复制下面命令，在终端中执行：

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/long-woo/install-fe-env/main/install-fe-env.sh)"
```

已实现安装，并配置国内镜像源：

- [x] brew
- [x] git
- [x] zsh
- [x] nodejs & npm
- [x] yarn
- [x] @vue/cli & @vue/cli-init
